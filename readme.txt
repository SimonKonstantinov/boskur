Программа реализована  для  операционных систем Linux и Windows
Программа реализует вывод информации о  всех сокетах указанного процесса.

Способ ввода 
	для  Linux(файл main.cpp)  ввести имя  процесса как  аргумент  командной  строки. Для  справки использовать аргумент "/?"
	для Windows(файл 1.cpp)  запустить программу; Нажать  клавишу 1; Ввести  имя  процесса. Для  справки использовать команду 2.
